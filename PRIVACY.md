# Privacy Policy for Quip

**Last Updated:** April 12, 2026

## Summary

Quip does not collect, store, or share any personal user data. Everything happens locally on your device.

## Data Collection

Quip **does not**:
- Collect or log your API key
- Collect or log generated comments
- Track your LinkedIn activity
- Send data to any server except OpenAI
- Use analytics or telemetry
- Store browsing history
- Create user profiles

## How Quip Works

1. Your OpenAI API key is stored locally in your browser (`chrome.storage.local`)
2. When you generate a comment, Quip sends the LinkedIn post content to OpenAI's API using YOUR key
3. OpenAI returns comment suggestions
4. Everything else stays on your device

## Third-Party Services

Quip sends post context to **OpenAI's API** (https://api.openai.com) for comment generation. You are responsible for your OpenAI API usage and must comply with OpenAI's Terms of Service.

See: https://platform.openai.com/docs/legal/terms-for-api-usage

## Open Source

Quip is open source on GitHub. You can audit the code to verify no data collection: https://github.com/muhammadtalhaishtiaq/quip
